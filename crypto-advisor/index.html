Come usare questo software

1. Dashboard — Vedi prezzi live, segnali tecnici (RSI, MACD, EMA, Bollinger) e il piano DCA ottimale per ogni coin.

2. Analisi AI — Inserisci la tua API key e premi il pulsante per ricevere un'analisi dettagliata da Claude con livelli di supporto/resistenza e consigli specifici.

3. Registro operazioni — Dopo ogni acquisto o vendita su Binance, registrala qui. Il software calcola automaticamente il tuo prezzo medio di carico e il P&L.

4. Portafoglio — Panoramica completa del tuo investimento: valore attuale, guadagni/perdite per ogni coin.

⚠ Questo software è uno strumento di analisi tecnica. Non costituisce consulenza finanziaria. Investi solo quello che puoi permetterti di perdere.
